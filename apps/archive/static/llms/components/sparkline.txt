# Sparkline

Lightweight inline SVG sparkline. No axes, no legend, no brewer. Embed in table cells, cards, or inline text.

## Import

```javascript
import { Sparkline } from '@rokkit/chart'
```

## Quick Start

```svelte
<!-- Numeric array -->
<Sparkline data={[12, 45, 23, 67, 34, 89]} />

<!-- Object array with field -->
<Sparkline data={rows} field="revenue" type="bar" color="danger" />

<!-- Area with custom size -->
<Sparkline data={rows} field="value" type="area" width={120} height={32} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `number[] \| Object[]` | `[]` | Series data |
| `field` | `string` | — | Field name when data is objects |
| `type` | `'line' \| 'bar' \| 'area'` | `'line'` | Chart type |
| `color` | `string` | `'primary'` | Semantic color role (`primary`, `success`, `danger`, etc.) |
| `width` | `number` | `80` | SVG width in pixels |
| `height` | `number` | `24` | SVG height in pixels |
| `min` | `number` | auto | Y domain minimum override |
| `max` | `number` | auto | Y domain maximum override |

## Notes

- Uses `--color-{role}-500` for stroke/bar fill, `--color-{role}-300` for area fill
- No `data-chart-*` attributes — standalone, not connected to any brewer
- Safe to embed anywhere inline
