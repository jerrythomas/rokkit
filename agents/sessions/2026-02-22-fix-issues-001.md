### Bug Fixes (High Priority)

- [x] 1. **Input text value binding bug** — Already fixed in commit `b660d747`. All input components now use `handleChange(event)` wrapper that extracts `event.target.value` before calling `onchange?.(value)`. Added regression test in `InputText.spec.svelte.js`.

This removes the visual gap that caused the "extra thick border" appearance.
