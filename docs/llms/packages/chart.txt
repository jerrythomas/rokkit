# @rokkit/chart

D3-powered composable chart components for Svelte 5. High-level chart components, a low-level declarative geom API, faceted and animated plots, inline sparklines, and a reactive crossfilter system for linked views.

## Install

```bash
npm install @rokkit/chart
```

## Continuous Color Scales

For heatmaps, hexbins, and other density visualizations, use sequential or diverging color scales:

```javascript
import { buildSequentialScale, buildDivergingScale } from '@rokkit/chart'

// Sequential: 0→100 mapped to a blue gradient
const seq = buildSequentialScale(data, 'value', { colorScheme: 'blues' })

// Diverging: -10→0→10 mapped to red-white-blue
const div = buildDivergingScale(data, 'delta', { colorScheme: 'rdbu', colorMidpoint: 0 })
```

Available sequential schemes: `blues`, `greens`, `oranges`, `purples`, `reds`, `viridis`, `inferno`, `magma`, `plasma`, `cividis`, `turbo`, `warm`, `cool`, and more.

Available diverging schemes: `rdbu`, `rdylbu`, `rdylgn`, `brbg`, `piyg`, `prgn`, `puor`, `spectral`.

Configure via `PlotSpec`:

```javascript
const spec = {
  data, x: 'day', y: 'hour', color: 'value',
  colorScale: 'sequential',    // 'categorical' | 'sequential' | 'diverging'
  colorScheme: 'viridis',
  geoms: [{ type: 'heatmap' }],
  legend: true
}
```

## Plot State (advanced)

`PlotState` is the reactive state manager used by `PlotChart`. It coordinates scales, colors, and geom registration.

```javascript
import { PlotState } from '@rokkit/chart'
```

## Presets

```javascript
import { createChartPreset, defaultPreset } from '@rokkit/chart'

const custom = createChartPreset({
  colors: ['blue', 'rose', 'amber'],
  opacity: { area: 0.4 }
})
```

## Brewer System (advanced)

Brewers transform raw data into chart-ready geometry. Extend to build custom chart types.

```javascript
import { ChartBrewer, CartesianBrewer, PieBrewer, QuartileBrewer, BoxBrewer, ViolinBrewer } from '@rokkit/chart'
```

| Class | Extends | Use for |
|-------|---------|---------|
| `ChartBrewer` | — | Base class — override `transform()`, add `$derived` mark properties |
| `CartesianBrewer` | `ChartBrewer` | X/Y cartesian charts with scales |
| `PieBrewer` | `ChartBrewer` | Pie/donut charts |
| `QuartileBrewer` | `ChartBrewer` | Shared quartile transform + scales for box/violin |
| `BoxBrewer` | `QuartileBrewer` | Adds `boxes` derived property |
| `ViolinBrewer` | `QuartileBrewer` | Adds `violins` derived property |
