# @rokkit/chart

> D3-powered composable chart components for Svelte 5. Build bar charts (with more chart types planned) using a declarative component API backed by a reactive ChartBrewer state manager.

Charts are composed from `Plot.*` components nested inside `Plot.Root`. The root manages dimensions, scales, and provides context to all children via `getContext('chart-brewer')`.

## Install

```bash
npm install @rokkit/chart
```

---

## Components

All components are exported under the `Plot` namespace:

```javascript
import { Plot } from '@rokkit/chart'
```

### `Plot.Root`

Container component. Manages responsive sizing, sets up ChartBrewer context, renders an SVG.

```svelte
<Plot.Root data={rows} width={600} height={400}>
  <Plot.Bar x="category" y="value" />
  <Plot.Axis type="x" field="category" />
  <Plot.Axis type="y" field="value" label="Revenue ($)" />
  <Plot.Grid />
  <Plot.Legend />
</Plot.Root>
```

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `array` | `[]` | Data rows array |
| `width` | `number` | `600` | Chart width in px |
| `height` | `number` | `400` | Chart height in px |
| `margin` | `object` | `{top:20,right:30,bottom:40,left:50}` | Margin on each side |
| `fill` | `string` | `null` | Background fill |
| `responsive` | `boolean` | `true` | Auto-resize to container width |
| `animationDuration` | `number` | `300` | Default transition duration in ms |
| `children` | `Snippet` | — | Child plot components |

### `Plot.Bar`

SVG bar series. Reads data and scales from `Plot.Root` context.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `x` | `string` | — | Data field for x-axis (category) |
| `y` | `string` | — | Data field for y-axis (value) |
| `fill` | `string` | — | Data field for color grouping |
| `color` | `string` | `'#4682b4'` | Fallback bar color |
| `opacity` | `number` | `1` | Bar opacity |
| `animationDuration` | `number` | `300` | Enter/update animation duration |
| `onClick` | `(data, event) => void` | — | Click handler for individual bars |

### `Plot.Axis`

X or Y axis with optional label and tick formatting.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'x' \| 'y'` | `'x'` | Axis direction |
| `field` | `string` | — | Data field this axis represents |
| `label` | `string` | `''` | Axis label text |
| `ticks` | `number` | — | Suggested tick count |
| `tickFormat` | `(value) => string` | — | Custom tick label formatter |
| `grid` | `boolean` | `false` | Render grid lines from this axis |

### `Plot.Grid`

Background grid lines derived from chart scales.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `'x' \| 'y' \| 'both'` | `'both'` | Which grid lines to render |
| `xTicks` | `number` | `null` | X tick count override |
| `yTicks` | `number` | `null` | Y tick count override |
| `color` | `string` | `'currentColor'` | Grid line color |
| `opacity` | `number` | `0.1` | Grid line opacity |
| `lineStyle` | `'solid' \| 'dashed' \| 'dotted'` | `'solid'` | Grid line style |

### `Plot.Legend`

Color legend with interactive filtering. Clicking legend items shows/hides corresponding bars.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `''` | Legend title |
| `align` | `'left' \| 'right' \| 'center'` | `'right'` | Horizontal placement |
| `verticalAlign` | `'top' \| 'middle' \| 'bottom'` | `'top'` | Vertical placement |
| `shape` | `'rect' \| 'circle'` | `'rect'` | Marker shape |
| `markerSize` | `number` | `10` | Marker size in px |
| `onClick` | `(selectedItems: any[]) => void` | — | Called when selection changes |

## ChartBrewer (programmatic API)

`ChartBrewer` is the reactive state manager that backs all `Plot.*` components. Accessible inside child components via `getContext('chart-brewer')`, or instantiated directly for headless/programmatic use.

```javascript
import { ChartBrewer } from '@rokkit/chart'

const brewer = new ChartBrewer({ width: 800, height: 400 })
brewer.setData(rows)
brewer.setFields({ x: 'month', y: 'revenue', color: 'region' })
brewer.createScales()
const bars = brewer.createBars()
const xAxis = brewer.createXAxis()
const legend = brewer.createLegend()
```

### Constructor options

| Option | Type | Default |
|--------|------|---------|
| `width` | `number` | `600` |
| `height` | `number` | `400` |
| `margin` | `object` | `{top:20,right:30,bottom:40,left:50}` |
| `padding` | `number` | `0.1` |
| `animationDuration` | `number` | `300` |

### Methods

| Method | Description |
|--------|-------------|
| `setData(data)` | Set chart data array |
| `setFields({ x, y, color })` | Set field mappings |
| `setDimensions({ width, height, margin })` | Update dimensions |
| `createScales()` | Generate D3 scales from data and fields |
| `createBars()` | Compute bar positions (x, y, width, height) |
| `createGroupedBars(data, fields, scales, options)` | Layout grouped bars |
| `createXAxis(options?)` | Generate x-axis tick data |
| `createYAxis(options?)` | Generate y-axis tick data |
| `createGrid(options?)` | Generate grid line data |
| `createLegend(options?)` | Generate legend items |
| `getData()` | Get current data array |
| `getFields()` | Get current field mappings |
| `getDimensions()` | Get current dimensions |
| `getScales()` | Get D3 scale objects |
| `getAnimationDuration()` | Get animation duration in ms |

## Utility functions

All utility functions are also exported directly for headless use:

```javascript
import {
  createDimensions, updateDimensions,
  createScales, getOrigin,
  createBars, filterBars, createGroupedBars,
  createXAxis, createYAxis, createGrid,
  createLegend, filterByLegend
} from '@rokkit/chart'
```

| Function | Description |
|----------|-------------|
| `createDimensions(width?, height?, margin?)` | Compute chart area with margins |
| `updateDimensions(current, overrides)` | Merge dimension updates |
| `createScales(data, fields, dimensions, options)` | Generate D3 band/linear scales |
| `getOrigin(scales, dimensions)` | Get origin point for axes |
| `createBars(data, fields, scales, options)` | Compute bar x/y/width/height |
| `filterBars(bars, activeItems)` | Filter bars by legend selection |
| `createGroupedBars(data, fields, scales, options)` | Layout grouped bars |
| `createXAxis(scales, dimensions, options)` | Build x-axis tick data |
| `createYAxis(scales, dimensions, options)` | Build y-axis tick data |
| `createGrid(scales, dimensions, options)` | Build grid line data |
| `createLegend(data, fields, scales, options)` | Build legend items array |
| `filterByLegend(data, activeItems)` | Filter data by selected legend items |

## Complete example

```svelte
<script>
  import { Plot } from '@rokkit/chart'

  const data = [
    { month: 'Jan', revenue: 4200, region: 'North' },
    { month: 'Jan', revenue: 3100, region: 'South' },
    { month: 'Feb', revenue: 5800, region: 'North' },
    { month: 'Feb', revenue: 4200, region: 'South' },
  ]
</script>

<Plot.Root {data} width={640} height={400}>
  <Plot.Bar x="month" y="revenue" fill="region" />
  <Plot.Axis type="x" field="month" />
  <Plot.Axis type="y" field="revenue" label="Revenue ($)" />
  <Plot.Grid direction="y" />
  <Plot.Legend title="Region" align="right" />
</Plot.Root>
```
