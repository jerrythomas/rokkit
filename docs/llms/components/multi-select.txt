# Rokkit MultiSelect Component

> Dropdown component for selecting multiple items with tag display.

MultiSelect extends Select for multi-item selection. Selected values are shown as removable tags inside the trigger. Supports grouping and field mapping.

## Quick Start

```svelte
<script>
  import { MultiSelect } from '@rokkit/ui'

  let items = ['JavaScript', 'TypeScript', 'Svelte', 'React', 'Vue']
  let value = $state([])
</script>

<MultiSelect {items} bind:value placeholder="Select languages..." />
<p>Selected: {value.join(', ')}</p>
```

## Object Items

```svelte
<script>
  import { MultiSelect } from '@rokkit/ui'

  const tags = [
    { id: 1, name: 'Frontend',  color: '#3b82f6' },
    { id: 2, name: 'Backend',   color: '#10b981' },
    { id: 3, name: 'DevOps',    color: '#f59e0b' }
  ]
  const fields = { value: 'id', label: 'name' }

  let value = $state([])
  let selected = $state([])
</script>

<MultiSelect {items} {fields} bind:value bind:selected />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `unknown[]` | `[]` | Data array (objects or primitives) |
| `fields` | `Record<string,string>` | `{}` | Field mapping |
| `value` | `unknown[]` | `[]` | Selected values array — use `bind:value` |
| `selected` | `unknown[]` | `[]` | Selected items — use `bind:selected` |
| `placeholder` | `string` | — | Placeholder when nothing selected |
| `size` | `'sm'|'md'|'lg'` | `'md'` | Size variant |
| `align` | `'start'|'end'` | `'start'` | Dropdown alignment |
| `direction` | `'up'|'down'` | `'down'` | Dropdown direction |
| `maxDisplay` | `number` | `3` | Max tags shown before count indicator |
| `disabled` | `boolean` | `false` | Disable the component |
| `icons` | `MultiSelectIcons` | — | Override state icons |
| `class` | `string` | `''` | Additional CSS classes |

## Field Mapping

Same as [Select](/llms/components/select.txt):

| Field | Default |
|-------|---------|
| `value` | `'value'` |
| `label` | `'label'` |
| `icon` | `'icon'` |
| `disabled` | `'disabled'` |
| `children` | `'children'` |

## Callbacks

| Callback | Signature | Description |
|----------|-----------|-------------|
| `onchange` | `(values[], items[]) => void` | Selection changed |

## Max Display

Collapse excess tags to a count indicator:

```svelte
<MultiSelect {items} bind:value maxDisplay={3} />
```

## Snippets

Supports `itemContent` and `groupContent` snippets (same as Select) for customizing option rendering in the dropdown. The trigger tag display is hardcoded — there is no `selectedValues` snippet.

```svelte
<MultiSelect {items} bind:value>
  {#snippet itemContent(proxy)}
    <span class={proxy.get('icon')}></span>
    <span>{proxy.label}</span>
  {/snippet}
</MultiSelect>
```

## Keyboard Navigation

Same as Select:

| Key | Action |
|-----|--------|
| `Enter/Space` | Open dropdown |
| `ArrowDown/Up` | Navigate options |
| `Enter` | Toggle focused option |
| `Escape` | Close dropdown |
| `Backspace` | Remove last selected pill |

## Data Attributes

| Attribute | Description |
|-----------|-------------|
| `data-multiselect` | Root element (also has `data-select` for theme compat) |
| `data-select-trigger` | Trigger button |
| `data-select-tags` | Tags container |
| `data-select-tag` | Individual selected tag |
| `data-select-tag-text` | Tag label text |
| `data-select-tag-remove` | Tag remove button |
| `data-select-count` | Count indicator when exceeding maxDisplay |
| `data-select-dropdown` | Dropdown container |
| `data-select-option` | Option items in dropdown |
| `data-open` | Open state |

## Import

```javascript
import { MultiSelect } from '@rokkit/ui'
```

## TypeScript Types

```typescript
interface MultiSelectProps {
  items?: unknown[]
  fields?: Record<string, string>
  value?: unknown[]
  selected?: unknown[]
  placeholder?: string
  size?: 'sm' | 'md' | 'lg'
  align?: 'start' | 'end'
  direction?: 'up' | 'down'
  maxDisplay?: number
  disabled?: boolean
  icons?: MultiSelectIcons
  class?: string
  onchange?: (values: unknown[], items: unknown[]) => void
  itemContent?: Snippet<[ProxyItem]>
  groupContent?: Snippet<[ProxyItem]>
}

interface MultiSelectIcons {
  opened?: string  // dropdown open arrow
  closed?: string  // dropdown closed arrow
  checked?: string // checkbox indicator on selected option
  remove?: string  // tag remove button
}
```
