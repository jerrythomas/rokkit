## Styles

Select and multi select should also be similar. on focus we should use secondary color for the underline if gradient is not possible. Since we have the input-root background wrapper we can use ::after selector to create a 1px height gradient background and use it as the underline. The label text is sorrectly capitalized, but size should be smaller. input text height including y padding does not match select component.
